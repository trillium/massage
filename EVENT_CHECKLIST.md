# Chair Massage Event Checklist

## Equipment

- [ ] Massage chair
- [ ] Fabric face cradles (×2 minimum)
- [ ] Face cradle covers / clean linens
- [ ] Hand sanitizer
- [ ] Wipes / cleaning spray
- [ ] Extension cord
- [ ] Music / portable speaker

## Marketing Materials

- [ ] Handbills
- [ ] Full-page advertisements
- [ ] Business cards

## Conditional Items

| Item                | Airbnb host event | SCaLE / trade show |
| ------------------- | :---------------: | :----------------: |
| Sandwich board sign |         ✗         |         ✓          |

## Before You Leave

- [ ] Chair fully packed and latched
- [ ] Face cradles counted and packed
- [ ] Marketing materials counted
- [ ] Phone charged
- [ ] Confirm address / parking

## At the Venue

- [ ] Scout setup location before unloading
- [ ] Set up chair, test height and angle
- [ ] Put out marketing materials
- [ ] Hand sanitizer accessible to clients

## Pack-Out

- [ ] Face cradle covers collected
- [ ] Chair wiped down
- [ ] All marketing materials accounted for (note how many distributed)
- [ ] Nothing left behind
