## Gear & Setup

- [ ] Massage chair
  - [ ] Face cradle covers
  - [ ] Spray cleaner
- [ ] A-frame sign (magnetic)
- [ ] Table or display surface? (do they provide one?)

## Marketing Materials

- [ ] 3D printed chair massage figures
  - [ ] Cards that go with 3D figures
- [ ] Business cards
- [ ] QR code printout → `/openclaw`
- [ ] QR code printout → `/openclaw-raffle`
- [ ] Are QR codes on the A-frame sign or separate printouts?
- [ ] Raffle prize — what is it? (free session? discount?)

## Digital Prep

- [ ] Generate raffle prize link (custom one time use link)
- [ ] Delete old custom redirect entries from previous event biz cards
- [ ] `/openclaw` landing page — reviewed and ready
- [ ] `/openclaw-raffle` — raffle form tested end to end
- [ ] `/openclaw-raffle-prize` — generated
- [ ] Test both QR codes resolve correctly on phone
- [ ] Raffle entries DB — cleared of old test data?

## Day-Of

- [ ] Arrive early — how early? Setup time?
  - [ ] 12:30PM if possible
- [ ] Take photos of setup for social media

## After Event

- [ ] Post event photos to social
- [ ] Draw raffle winner
  - [ ] All non-raffle winners should get openclaw-free-30-upgrade (60 -> 90, 90 -> 120, etc)
- [ ] Follow up with leads / new contacts
- [ ] Debrief — what worked, what to change next time
