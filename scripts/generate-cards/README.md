# generate-cards

Python scripts for producing print-ready business card PDFs with unique QR codes.

## Dependencies

```bash
pip3 install pymupdf pillow
brew install librsvg  # for rsvg-convert (SVG→PDF conversion)
```

## Scripts

### generate-cards.py — Individual card PDFs

Takes a 2-page PDF template (front + back) and a batch of QR SVGs, produces one PDF per card.

```bash
# Default: uses print/business-cards/template-v3.pdf, prefix BC-
python3 scripts/generate-cards/generate-cards.py

# Custom template and prefix
python3 scripts/generate-cards/generate-cards.py \
  --template print/templates/MassageCardV3_bc02.pdf \
  --prefix BCR01- \
  --type business-card-red

# Limit count (useful for test runs)
python3 scripts/generate-cards/generate-cards.py --prefix BCR01- --count 5
```

**Template requirements:**

- Page 1: front — used as-is (no modification)
- Page 2: back — must contain a solid green `#00EE00` square; the script detects it by pixel position and replaces it with the QR code SVG

**QR codes:**

- Source: `print/qr/{prefix}*.svg` (generated by `pnpm tsx scripts/generate-pdf-card.ts --prefix=BCR01- --count=50`)
- Output: `print/{date}_{type}/card-{slug}.pdf` — one PDF per slug, 2 pages each

**Color scheme:**

- QR border color pulled from `lib/qr/print-colors.json` (`containerBg` field)
- Current scheme: red `#dc2626` + gold `#ffbd4a`

### generate-print-sheet-business-cards.py — Gang sheet (CC-228 cutter)

Takes individual card PDFs and tiles them into a print sheet for the CC-228 card cutter (10-up, 2×5 duplex).

```bash
python3 scripts/generate-cards/generate-print-sheet-business-cards.py
```

---

### generate-print-sheet.py — Gang sheet (handbills)

6-up duplex handbill sheet (2×3), long-edge flip.

```bash
python3 scripts/generate-cards/generate-print-sheet.py \
  --pdf print/templates/BACK_Handbill_Business_Card.pdf
```

---

### generate-screen-pdf.py — Single QR display card

Large-format single QR code on a letter-size page (for screen/counter display).

```bash
python3 scripts/generate-cards/generate-screen-pdf.py screen_a1b2c3d4
```

### gang_sheet/

Pipeline that generates gang sheets in multiple paper sizes (8.5×11, 11×17, 12×18, 13×19).

```bash
python3 scripts/generate-cards/gang_sheet/gang-sheet-business-cards.py
```

## BCR01 Workflow (current active run)

**Scope:** BCR01 — 50 red-scheme business cards (arms crossed portrait variant)

**Front:** `portrait_arms_crossed_burgundy.jpg` (left, square) + `Let the Spa Come to You.pdf` (right panel)
**Back:** `Downloads/MassageCardV3_bc02 (4).pdf` — red scheme, green QR placeholder

**Step 1 — Generate QR SVGs** (done):

```bash
pnpm tsx scripts/generate-pdf-card.ts --prefix=BCR01- --count=50
# Output: print/qr/BCR01-*.svg (50 unique slugs)
```

**Step 2 — Generate card PDFs:**

```bash
python3 scripts/generate-cards/generate-cards.py \
  --template "print/templates/MassageCardV3_bc02.pdf" \
  --prefix BCR01- \
  --type business-card-red
```

**Step 3 — Gang sheet for print shop:**

```bash
python3 scripts/generate-cards/gang_sheet/gang-sheet-business-cards.py
```

## Scope Registry

Scopes map QR slugs to redirect destinations. Defined in `lib/qr/scope-defaults.ts`.

| Scope     | Destination                     | Cards                             |
| --------- | ------------------------------- | --------------------------------- |
| BC01      | trilliummassage.la              | legacy teal                       |
| BC02–BC06 | trilliummassage.la              | red scheme variants (50 each)     |
| BCR01     | trilliummassage.la              | red scheme, arms crossed portrait |
| HB01      | /blog/airbnb-host-promo-2026-03 | handbills                         |

Redirects resolve at `/rd/{slug}` via `app/rd/[code]/route.ts`.
